## How to install neovim

- macOS  
  `brew install neovim`

- Ubuntu  
  `sudo apt install neovim`

- RHEL *This is not recommended  
  `sudo yum install neovim`

- Install from git  
  need cmake
  ```
  git clone https://github.com/neovim/neovim.git
  cd neovim
  make
  make install
  ```

- check install for nvim  
  `nvim --version`
