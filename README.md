## How to install neovim

- macOS  
  `brew install neovim`

- Ubuntu  
  `sudo apt install neovim`

- RHEL  
  `sudo yum install neovim`  
  This is not recommended

- Install from git
  ```
  git clone https://github.com/neovim/neovim.git
  cd neovim
  make
  make install
  ```
